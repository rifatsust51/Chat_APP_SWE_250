# Chat_APP_SWE_250
Creating a real time chat app as an academic project


# These feature will be available on the project


1. User Authentication
Sign up / Login (Email or Phone)

Save name & profile picture

2. One-to-One Messaging
Send and receive messages in real-time

Show timestamp on messages

3. Online/Offline Status
Show if a user is online or their last seen time

4. Typing Indicator
Show "Typing..." when the other person is typing

5. Push Notifications
Get notified when a new message arrives

6. Media Sharing 
Send and view images

7. Chat List
See a list of all conversations

Last message and timestamp preview

8. Logout
Securely logout and clear user session
